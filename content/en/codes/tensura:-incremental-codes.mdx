## Troubleshooting Code Issues

If you find that a code is not working, there are several common reasons why this might occur:

*   **Expired Status:** The developers may have deactivated the code recently. We update our list frequently, but some codes have very short lifespans.
*   **Case Sensitivity:** Ensure that "SLIMETIME" isn't entered as "slimetime." The game engine treats these as different strings.
*   **Spaces:** Extra spaces at the beginning or end of the code (often caused by copying and pasting) will cause the redemption to fail.
*   **One-Time Use:** Most **tensura: incremental codes** can only be redeemed once per account. If you have used it in the past, it will return an "Already Redeemed" error.
