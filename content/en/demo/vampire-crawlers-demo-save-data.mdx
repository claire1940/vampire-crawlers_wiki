## Managing Your Vampire Crawlers Demo Save Data

For the majority of players on Steam, the **vampire crawlers demo save data** is handled via the Steam Cloud. However, if you are playing on a Steam Deck or prefer manual backups, knowing the file location is vital. The save files are typically located in the `AppData/LocalLow` directory under the developer's folder.

### Manual Backup Steps
1. Navigate to `%USERPROFILE%\AppData\LocalLow\VampireCrawlers\`.
2. Locate the `SaveData.vcd` file (or similar extension).
3. Copy this file to a secure folder before installing the full game.
4. If the full game does not detect your progress, paste the file into the new installation's save folder.

## Troubleshooting Save Synchronization

Occasionally, players may find that their **vampire crawlers demo save data** does not appear in the full version. This is usually due to a version mismatch or a conflict with Steam Cloud. If you find your character list empty or your gold set to zero, do not panic. The data is likely still on your hard drive.

> **⚠️ Warning:** Do not click "New Game" or "Reset Progress" in the settings menu if your data is missing, as this may overwrite your cloud save with a blank file.

1. **Check Version Numbers:** Ensure both the demo (if still installed) and the full game are updated to the latest 2026 patches.
2. **Verify Integrity:** Right-click the game in Steam > Properties > Installed Files > Verify Integrity of Game Files.
3. **Steam Cloud Conflict:** If prompted with a "Cloud Sync Conflict," always choose the "Local File" option if it has a more recent timestamp or a larger file size.

## FAQ

**Q: Can I transfer my vampire crawlers demo save data from a console to PC?**

A: Currently, cross-platform save transfers are not officially supported for the 2026 launch. Save data is restricted to the platform (Steam, Xbox, or Switch) where the demo was played.

**Q: Will I miss any achievements if I use my demo save?**

A: No. The game is designed to "retro-unlock" achievements. Upon loading your demo save, any achievements you earned during the demo period should trigger simultaneously on your Steam or console profile.

**Q: What happens if I delete the demo before installing the full game?**

A: As long as Steam Cloud was enabled, your **vampire crawlers demo save data** is safely stored on Valve's servers. If you played offline, ensure you do not delete the `LocalLow` folder mentioned in the management section of this guide.

**Q: Are there any demo-exclusive items that carry over?**

A: While there are no "exclusive" items that disappear after the demo, players who completed certain milestones in the demo may receive a small bonus of "Bonus Coins" (typically 500-1000) upon their first login to the full 2026 version.
