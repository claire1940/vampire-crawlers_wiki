## O Mecanismo de Combo: Como Quebrar o Jogo

O núcleo de qualquer build "apelona" em Vampire Crawlers é o sistema de combo. Toda vez que você joga uma carta com um custo que segue o custo da carta anterior (ex: jogar uma de custo 1 após uma de custo 0), seu contador de combo aumenta. Este contador aplica um multiplicador ao seu dano e efeitos.

| Sequência de Custo da Carta | Efeito do Multiplicador | Dica de Estratégia |
|---|---|---|
| **0 → 1 → 2** | Multiplicador Base | Use cartas de custo 0 como a **Magic Wand** para iniciar a corrente. |
| **0 → 1 → 2 → 3 → 4** | Multiplicador Avançado | Requer um tamanho de mão grande para garantir que você tenha todos os custos. |
| **Wild/Friendship** | Extensão de Combo | Estas cartas aumentam o contador sem exigir um custo específico. |

> **💡 Dica:** Sempre priorize melhorias de "Tamanho da Mão" (Hand Size). Uma mão maior torna estatisticamente muito mais provável que você consiga completar uma corrente de combo de 0 a 4 a cada turno.

## Sinergias de Arcana que Quebram o Jogo

*   **Combos Don't Clear (Combos não resetam):** Esta Arcana permite que você mantenha seu contador de combo entre os turnos. Se você terminar um turno no Combo 5, sua primeira carta no próximo turno começa no Combo 6. Isso leva a números de dano astronômicos.
*   **Mana Siphon (Sifão de Mana):** Aumenta permanentemente sua mana máxima para cada 100 cartas jogadas. Em partidas longas, isso permite que você jogue seu deck inteiro a cada turno.
